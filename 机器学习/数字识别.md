## 准备数据：将图像转换为向量

将32x32的二进制图像矩阵转换为1x1024的向量
```
import numpy as np
def img2vector(filename):
    returnVect = np.zeros((1,1024))
    fr = open(filename)
    for i in range(32):
        lineStr = fr.readline()
        for j in range(32):
            returnVect[0,32*i+j] = int(lineStr[j])
    return returnVect
```
