##### 更新
```
sudo apt-get update
```
---

##### 删除
```
rm -R -v <文件/文件夹>
```
---

##### 复制
```
cp -R -v <源文件/文件夹> <目标文件/文件夹>
```
---

##### 权限更改
```
chmod -R -v 777 <文件/文件夹>
```
---

##### 解压tgz文件
```
tar   zxvf    <.tgz文件>  -C  <指定目录>
```
---

##### 清空回收站
```
cd ~/.local/share/Trash/

rm -rf files/*
```

##### 查找字符串
在命令模式下，输入/你要查找的字符

按下回车，可以看到vim把光标移动到该字符处

再按n（小写）查看下一个匹配

按N(大写）查看上一个匹配

##### 查看命令是否执行完毕

`echo $?` 命令，返回0，则说明上个执行命令已经成功执行
