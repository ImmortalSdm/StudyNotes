
Ubuntu主机下载NVIDIA SDK manager

https://developer.nvidia.com/nvidia-sdk-manager

安装
```
sudo apt-get update
sudo apt-get upgrade
sudo apt install ./<sdk文件名>.deb
```



启动NVIDIA SDK manager
```
sdkmanager
```

若报错：no SDKs are available for your
```
sudo updatedb
```
