启动节点管理器
```
roscore
```

启动一个节点
```
rosrun <package name> <executable name>

#例如
rosrun turtlesim turtlesim_node
```

列出活动节点
```
rosnode list
```

查看节点
```
rosnode info <节点>
```

列出话题
```
rostopic list
```

查看某个消息类型的具体内容
```
rosmsg show
```

## 生成并构建ROS功能包

```
cd ~/catkin_ws/src

#在src中生成/ros_robotics目录，包含package.xml和CMakeLists.txt
catkin_create_pkg ros_robotics

cd ~/catkin_ws

catkin_make
```
